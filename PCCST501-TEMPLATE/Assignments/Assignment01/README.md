## Title
How the Internet Delivers a Web Page Using TCP/IP Protocol Stack (Wikipedia Example)
## Objective
To understand how a web page is delivered using the TCP/IP protocol stack and application-layer protocols.
## Solution
The browser uses DNS to obtain Wikipedia's IP address, sends an HTTP request through TCP/IP, and receives the webpage reliably from the server.
## Output
The requested Wikipedia webpage is successfully displayed in the browser with reliable and secure communication.
## Conclusion
DNS, HTTP, TCP, cookies, and the client–server architecture work together to provide fast, reliable, and secure web access.
